幫我做一個本地網頁項目，名字叫「霓虹彈力球實驗室」。 使用 HTML、CSS、JavaScript 實現，不需要後端，本地打開就能運行。

頁面要求：

  - 整體風格： 深色賽博科技感。
  - 中間區域： 全屏 Canvas 動畫。
  - 元素： 頁面裡有 50 個霓虹發光小球。
  - 基礎行為： 小球會隨機運動，碰到邊界會反彈，小球之間要有簡單碰撞效果，運動時帶有拖尾殘影。
  - 視覺互動： 小球之間距離較近時，自動繪製發光連線；鼠標移動時，小球會被鼠標輕微吸引。
  - 點擊行為： 點擊頁面時，在點擊位置生成 10 個新小球。
  - 數據監測（左上角顯示實時數據）：
      - 當前小球數量
      - FPS
      - 碰撞次數
      - 鼠標坐標
  - 控制功能（右下角放 3 個控制按鈕）：
      - 暫停 / 繼續
      - 重置
      - 爆炸效果（點擊後，所有小球從中心向四周散開，視覺效果要有明顯衝擊力，適合短視頻錄屏展示）。

技術要求：

  - 代碼分成 index.html、style.css、script.js 三個文件。
  - 最後請告訴我該如何運行。
